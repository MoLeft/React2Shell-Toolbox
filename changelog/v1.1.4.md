# v1.1.4 更新日志

发布日期：2025-12-20

## 🎨 界面优化

### 批量验证页面布局优化

- **搜索结果标题栏优化**：标题栏现在横跨整个宽度，布局更清晰
- **筛选面板优化**：只在搜索后显示筛选面板，避免空白区域
- **表头固定**：表格表头现在固定在顶部，滚动时始终可见
- **高度统一**：筛选条件标题和表格表头高度统一为 48px，视觉更整齐
- **滚动条优化**：
  - 修复了全局滚动条不显示的问题
  - 表头区域不显示滚动条，只在内容区域显示
  - 滚动条样式更美观

## 🐛 Bug 修复

### 数据加载优化

- **表格加载遮罩优化**：数据加载完成后立即隐藏遮罩，不再等待图标和状态检查
- **元数据异步加载**：图标和状态检查改为异步加载，不阻塞页面显示

### 批量验证功能修复

- **统计信息保留**：修复了暂停后恢复批量验证会清空统计信息的问题
- **状态管理优化**：验证完成时正确保留统计数据

### 自动加载功能修复

- **按钮状态同步**：
  - 修复了按钮状态与后台加载状态不一致的问题
  - 点击暂停按钮后立即响应，不再等待页面加载完成
- **暂停功能修复**：
  - 修复了暂停后仍继续加载数据的问题
  - 使用简单的标志位代替 AbortController，提高可靠性
- **状态重置**：新搜索时自动重置自动加载状态
- **按钮图标优化**：
  - idle 状态：显示下载图标
  - paused 状态：显示播放图标
  - loading 状态：显示暂停图标
  - completed 状态：显示完成图标
  - error 状态：显示错误图标

### 表格显示优化

- **排序功能移除**：移除了表头的排序箭头，界面更简洁
- **表头背景**：表头使用不透明的浅灰色背景，滚动时不会透明

## 🔧 技术改进

- **样式优化**：整理了 flex 布局链，确保所有容器正确设置 `min-height: 0`
- **滚动条强制显示**：添加全局样式确保滚动条在所有浏览器中正常显示
- **代码清理**：移除了冗余的状态设置和样式

## 📝 国际化

- 添加了 `batch.autoLoad.pause` 翻译（中英文）
